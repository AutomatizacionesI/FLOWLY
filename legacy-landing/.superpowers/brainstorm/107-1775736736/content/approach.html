How far do we redesign synara.ai?

All 3 approaches keep your existing dark glassmorphism style. The question is how much new structure we add.

A

Surgical Fixes

What changes: Replace round stats (85%, 95%) with credible specific numbers (82%, 94%, etc). Sharpen all copy. Place the 4 corporate images into existing sections as framed visuals. Add the 4 AI avatars as a proper testimonials row inside the Impact section.

New HTML elements: ~20 lines — just image frames and 4 quote cards.

Pros

  • Lowest risk, fastest to implement
  • No layout shifts
  • Keeps existing SEO structure

Cons

  • Doesn't add new trust signals
  • Testimonials feel tacked on
B

Balanced Overhaul (Recommended)

What changes: Everything in A, PLUS: add a full Testimonials section (4 quote cards with ai_avatar_1–4, company role, star rating) between Impact and Stats. Redesign the Impact section so each "Con synara.ai" row has a supporting corporate image (opt_robotic_process.png, opt_business_dashboard.png). Hero right-side gets opt_office_team.png as a floating card for human trust.

New HTML elements: ~80 lines — new section + enhanced impact cards.

Pros

  • Maximum credibility uplift
  • Avatars used as real social proof
  • Corporate images build trust exactly where needed
  • Still matches your existing visual system

Cons

  • Testimonial quotes need to be written/confirmed
  • More implementation work
C

Full CRO Rebuild

What changes: Everything in B, PLUS: replace the hero right-side orbit animation with a full glassmorphism "floating dashboard" panel (opt_business_dashboard.png in a macOS-style window frame). Add a Results / Case Studies section with 2–3 cards (opt_server_room.png, opt_office_team.png) each showing industry + specific metric achieved. Move CTA button above the fold in the hero.

New HTML elements: ~200 lines — hero redesign + new sections.

Pros

  • Highest-converting layout
  • Every image used strategically
  • Looks like a funded startup's site

Cons

  • More risk of breaking existing animations
  • Hero redesign is significant work
  • Needs case study content
